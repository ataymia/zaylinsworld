# zaylinsworld

Private testing browser sandbox for **Zaylin's World**.

## What this prototype includes

- Kid-friendly open-world city grid exploration
- Character customization (color + style)
- Car toggling with simple traffic simulation
- Jobs, housing purchase, school progression, and stat growth
- NPC memory snippets based on player actions
- Consequence loop via heat/fines and reputation changes
- Optional "Monster Mode"
- Frostbox jewelry customization
- Block Supply gear customization
- Stylized GTA-inspired progression loop for private testing

## Run locally

From `/home/runner/work/zaylinsworld/zaylinsworld/ataymia/zaylinsworld`:

```bash
python3 -m http.server 8000
```

Open `http://localhost:8000/index.html`.
